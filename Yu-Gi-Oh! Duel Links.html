Description

Yu-Gi-Oh! Duel Links is a free-to-play, digital collectible card game developed by Konami for the iOS, Android and Microsoft Windows platforms, based on Yu-Gi-Oh! Trading Card Game. After an initial beta period, the game was first released in Japan on November 17, 2016, and then released to the rest of the world on January 11, 2017.[1] The Windows version was released worldwide via Steam on November 17, 2017.
